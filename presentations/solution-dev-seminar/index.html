# 내 일하는 방식 프로파일링 접근 가능한 대화와 작업 히스토리, 프로젝트 파일을 찾아 분석해줘. 내가 지시하고 · 검증하고 · 수정하는 방식에서 반복되는 패턴을 찾아, - 나의 AI 협업 성향과 일하는 방식 - skill로 만들 반복 절차 - workflow로 묶을 과정 - 도구를 붙이면 좋은 지점 을 근거와 함께 추천해줘. 한 번에 구축하지 말고, 가장 작은 후보부터 하나씩 적용해보자.
