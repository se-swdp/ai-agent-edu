- 텐션 곡선: 낙관(①②) → 현실(③) → 종합(④). 14장 소결("낙관의 곡선")이 ③으로 넘어가는 브릿지, 16장 Ronacher의 "토큰 팩토리" 냉소가 비용 파트 진입 브릿지.
