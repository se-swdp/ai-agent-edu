solution-dev-seminar(44슬라이드, 라이트 인포그래픽)의 노트필기 리워크.
손글씨 노트 스타일 이미지 35장을 한 장 = 한 슬라이드로 넘기는 이미지 덱.

- `index.html` — 이미지 뷰어 셸 (키보드 ←/→/Space/Home/End, `F` 전체화면, 터치 스와이프, `#N` 해시 딥링크).
  슬라이드는 네이티브 1672×941을 넘지 않게 표시(업스케일 블러 방지) + 사방 여백(2.6vh/3vw).
- `NN-slug.webp` — 슬라이드 본체 35장 (1672×941, **q95**, 총 ~7.8MB — q86은 손글씨 획이 물러져 격상)
- `src-png/` — 원본 PNG 보관 (git/배포 제외 대상, firebase ignore: `**/src-png/**`)
